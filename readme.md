# Code Dirt Checker

## 概要

AIとかにぐっちゃぐちゃに荒らされたプロジェクトを修正するのクソめんどくさい。やだ。あと過去の自分が実装した冗長なコード見返して読み解くのやだ。

少しでも楽にコードのゴミや汚れをお掃除できないか模索するためにこのプロジェクト作った。

「Code Dirt Checker」は、「コードの埃チェッカー」って意味。

## インストール

現状は標準ライブラリのみで動作するため、追加のインストールパッケージはありません。
Python環境（3.x系推奨）があれば動作します。

```bash
git clone https://github.com/IT-Engineer-K/Code-Dirt-Checker.git
cd CodeDirtChecker
```

## 使い方

対象のファイル、またはプロジェクトのディレクトリを指定して実行します。ディレクトリを指定した場合は、配下のファイルを再帰的にチェックします。

### 基本的な使い方

```bash
# 1つのファイルをチェックする場合
python check.py sample.txt

# プロジェクトディレクトリ全体をチェックする場合
python check.py sample/
```

### オプション指定
第2引数で「最小一致行数」を指定できます（デフォルトは4行）。

```bash
# 5行以上の重複ブロックをチェック
python check.py sample/ 5
```
